Part 1: Basic Iterators

เขียนฟังก์ชันที่รับ array ของตัวเลข แล้วใช้ forEach() พิมพ์ตัวเลขแต่ละตัวเป็นเลขยกกำลังสอง

ใช้ map() สร้าง array ใหม่ที่มีค่าตัวเลขใน array เดิม + 10

ใช้ filter() เลือกตัวเลขที่เป็นเลขคู่จาก array

Part 2: Search & Find

ใช้ find() หา element แรกที่มีค่าน้อยกว่า 0

ใช้ findIndex() หา index ของตัวเลขที่มากกว่า 100

ใช้ some() และ every() ตรวจสอบว่าใน array

Part 3: Reduce

ใช้ reduce() หาผลรวมของ array ตัวเลข

ใช้ reduce() หา ค่าสูงสุด ใน array

Part 4: Stack & Queue

สร้าง array ว่างแล้วใช้ push() ใส่สมาชิกทีละตัว

ใช้ pop() เอาค่าสุดท้ายออกมา และแสดง array หลังจากนั้น

ใช้ unshift() ใส่สมาชิกเพิ่มด้านหน้า

ใช้ shift() ลบสมาชิกตัวแรกออก

Part 5: Subarray

ใช้ slice() ดึงบางส่วนของ array (เช่น index 2–4)

ใช้ splice() แทรกสมาชิกใหม่เข้าไปกลาง array

ใช้ fill() ใส่ค่าเดิม (เช่น 0) ลงใน array

Part 6: Sorting & Searching

ใช้ includes() ตรวจสอบว่ามี "cat" อยู่ใน array หรือไม่

ใช้ reverse() กลับลำดับ array

ใช้ sort() เรียงเลขจากน้อยไปมาก และเรียง string ตามตัวอักษร

Part 7: Combine & Convert

ใช้ concat() รวมสอง array เข้าด้วยกัน

ใช้ join() แปลง array ของ string ให้กลายเป็นประโยคเดียว โดยเว้นวรรคระหว่างคำ
